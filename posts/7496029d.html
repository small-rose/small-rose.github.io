Matery 添加加载动画

Matery 添加过渡的加载动画，对于加载速度较慢是一种友好的解决方式。

起初我使用大佬的方案：Matery 主题添加 loading-pages

开始也正常使用了一段时间，今天发现加载动画不见了？

然后安装大佬的方案再来一遍，显示不出来，其实我并没有改什么，就是改下停留时间，好不容易显示出来了，但是我发现原本在 <head> 区的代码跑到了 <body> 区，其实大佬的方案对于使用完全没有任何问题。

可以继续使用。

那么我为什么要折腾这篇文章呢？

因为我是个追求完美的人，好吧，就这么说说，信不信不重要了~

反正就是想代码显示的正常一点，其实是担心会不会影响搜索什么的…

所以就按照大佬代码，变换了一下结构。

1、在 /layout/_partial/ 目录新建一个 loading-pages.ejs 内容如下：

如果你发现了爆红什么的，不要担心。

<style type="text/css" lang="css">
    #loading-container{
        position: fixed;
        top: 0;
        left: 0;
        min-height: 100vh;
        width: 100vw;
        z-index: 9999;
        display: flex;
        flex-direction: column;
        justify-content: center;
        align-items: center;
        background: #FFF;
        text-align: center;
        /* loader页面消失采用渐隐的方式*/
        -webkit-transition: opacity 1s ease;
        -moz-transition: opacity 1s ease;
        -o-transition: opacity 1s ease;
        transition: opacity 1s ease;
    }
    .loading-image{
        width: 120px;
        height: 50px;
        transform: translate(-50%);
    }

    .loading-image div:nth-child(2) {

    @-webkit-keyframes pacman-balls {75% {opacity: .7}to {transform: translate(-100px, -6.25px)}}

    @keyframes fadeInDown{0%{opacity:0;-webkit-transform:translate3d(0,-100%,0);}}
 </style>
 <script>
(function () {
    const loaded = function(){
       setTimeout(function(){
            const loader = document.getElementById("loading-container");
            loader.className="fadeout" ;//使用渐隐的方法淡出loading page
            // document.getElementById("body-wrap").style.display="flex";
            setTimeout(function(){
                loader.style.display="none";
            },2500); 
        },1000);//强制显示loading page 1s  
    };
    loaded();
})()
</script>

2、在主题的 layout 目录找到 layout.ejs 文件

在 <body> 附近添加内容后如下：

<!DOCTYPE HTML>
<html lang="<%= config.language %>">
<%- partial('_partial/head') %>

 <% if (theme.loadingPages.enable) { %>
    <%- partial('_partial/loading-pages') %>
 <% } %>
<body>
    <% if (theme.loadingPages.enable) { %>
        <div id="loading-container">
             <p class="loading-text">嘘~  正在从服务器偷取页面 . . . </p> 
             <div class="loading-image">
                 <div></div>
                 <div></div>
                 <div></div>
                 <div></div> 
                 <div></div>
             </div>
        </div>
    <% } %>

    <%- partial('_partial/header') %>
    ...

3、添加开关配置

loadingPages:
  enable: true

enable 设置为 true 表示开启加载动画， false 表示关闭加载动画。

4、大功告成

hexo clean && hexo g && hexo s

看看效果吧~
