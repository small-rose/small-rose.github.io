关于网站更换字体，还是独立出来，并不是所有的人都有这种需要。

一、CSS 字体支持

CSS支持字体样式，通过@font-face属性在网页中嵌入自定义字体，主流的浏览器都支持这种方式，同时CSS字体属性还可以定义字体，加粗，大小，文字样式等。

默认支持:

body{
    font-family: "serif","sans-serif","cursive","fantasy","monospace";
}
body{
    font-family: "sans-serif,monospace,微软雅黑,黑体,宋体";
}
<body>

我喜欢的写法：

.myfont{
    font-family:"微软雅黑","黑体","宋体";
}

HTML 使用

<div class="myfont"> Hello World! 这个网站很酷 zhangxiaocai.cn </div>

二、字体文件格式

web字体常见格式

.ttf

TrueType (.ttf) 是Windows和Mac系统最常用的字体格式，其最大的特点就是它是由一种数学模式来进行定义的基于轮廓技术的字体，这使得它们比基于矢量的字体更容易处理，保证了屏幕与打印输出的一致性。同时，这类字体和矢量字体一样可以随意缩放、旋转而不必担心会出现锯齿。

.eot

EOT – Embedded Open Type，嵌入字体格式（EOT）是微软开发的一种技术，允许 OpenType 字体嵌入到网页并可以下载至浏览器渲染，浏览器根据 CSS 中 @font-face 的定义，下载，渲染这种 .EOT 后缀的字体文件。这些文件只在当前页活动的状态下，临时安装在用户的系统中。

.otf

OpenType是一种可缩放字型（scalable font）电脑字体类型，采用PostScript格式，是美国微软公司与Adobe公司联合开发，用来替代TrueType字型的新字型。这类字体的文件扩展名为.otf，类型代码是OTTO，现行标准为OpenType 1.4。OpenType最初发表于1996年，并在2000年之后出现大量字体。它源于微软公司的TrueType Open字型，TrueType Open字型又源于TrueType字型。OpenType font包括了Adobe CID-Keyed font技术。Adobe公司已经在2002年末将其字体库全部改用OpenType格式。

.woff

相对于 TrueType 和 OpenType ，WOFF（WOFF – Web Open Font Forma是一种Web开发字体格式）是一种专门为了 Web 而设计的字体格式标准，它并不复杂，实际上只是对于 TrueType / OpenType 等字体格式的封装，并针对网络使用加以优化：每个字体文件中含有字体以及针对字体的元数据（Metadata），字体文件被压缩，以便于网络传输，并且不包含任何加密或者 DRM 措施。包括 Adobe、 Lino Type、Monotype 在内的几乎所有主要的字体供应商都加入到支持 WOFF 的行列中来

.svg

顾名思义，就是使用SVG(Scalable Vector Graphics) 技术来呈现字体SVG Fonts ，还有一种gzip压缩格式的SVG字体.svgz。SVG可缩放矢量图形（Scalable Vector Graphics）是基于可扩展标记语言（XML），用于描述二维矢量图形的一种图形格式。SVG由W3C制定，是一个开放标准。SVG严格遵从XML语法，并用文本格式的描述性语言来描述图像内容，因此是一种和图像分辨率无关的矢量图形格式。SVG可以使你设计的网页可以更加精彩细致，使用简单的文本命令，SVG可实现色彩线性变化、路径、自定义字体、透明效果、滤镜效果等各式常见的图形图像效果。

6、自定义引入

如果想使用自定义的字体，首先引入

@font-face {    
   font-family: 'myfont';    
   src:url('字体文件1.woff'),  
   url('字体文件2.ttf'),    
   url('字体文件3.eot');
}

body{
    font-family: 'myfont' ;
}

三、字体使用步骤

字体使用方式：

1、下载字体

可以在字体网站下载自己喜欢的字体：http://www.fonts.net.cn/

下载之后，一般是ttf格式，有些下载包是各种格式都有。

注意，字体的使用，要注意版权问题，不要使用未经许可的字体，一般来说，非商用没有太大问题，下载的时候注意是否使用付费。

文末整理了部分开源免费字体，仅供参考。

2、引入字体

引入字体也很简单，在css 文件引入即可。


@font-face{
    font-family: 'hl';
    src: url('https://cdn.jsdelivr.net/gh/small-rose/small-rose.github.io/box/font/hl.ttf');
}

@font-face{
    font-family: 'ming';
    src: url('https://cdn.jsdelivr.net/gh/small-rose/small-rose.github.io/box/font/GuDianMingChaoTi.otf');
}

针对元素使用

/* 针对全局的使用 */
body{
    font-family: 'YaHei';
}

/* 针对元素的class属性里包含 mytitle 的使用 */
.mytitle {
    font-family: 'ming' !important;
}

/* 针对元素的id属性等于 main-content 的使用 */

#main-content{
 font-family: '仿宋',"Segoe UI",Roboto,Oxygen-Sans,Ubuntu,Cantarell,"Helvetica Neue",'sans-serif';
}

.diyFont{
    font-family: 'hl';
}

根据样式选择器使用使用即可。

如果引入字体路径没有问题，却无法显示

3、样式引用

<html>
<body>
    <div class='diyFont'><span>我是标题</span></div>
    <div id="main-content">
        我是文章内容,使用
    </div>
    <div>
        我使用全局默认的字体
    </div>
</body>
</html>

4、字体转换

如果你确认引入路径绝对正确，复制路径到浏览器执行，看看会不会出现下载提示，有下载提示或者显示说明正确。

如果不能显示可以尝试字体转换。

在线的字体转换工具：

  • https://www.fontke.com/tool/convfont/
  • http://www.cuvou.com/wizards/ttf2eot.cgi
  • http://onlinefontconverter.com/
  • https://www.fontsquirrel.com/tools/webfont-generator
  • https://everythingfonts.com/font-face

多种格式引入参考：

@font-face{
    font-family: 'ming';
    src: url('https://cdn.jsdelivr.net/gh/small-rose/small-rose.github.io/box/font/GuDianMingChaoTi.otf');
    src: url('https://cdn.jsdelivr.net/gh/small-rose/small-rose.github.io/box/font/GuDianMingChaoTi.eof'), url('https://cdn.jsdelivr.net/gh/small-rose/small-rose.github.io/box/font/GuDianMingChaoTi.woff'), url('https://cdn.jsdelivr.net/gh/small-rose/small-rose.github.io/box/font/GuDianMingChaoTi.ttf'),/*non-IE*/
      url('https://cdn.jsdelivr.net/gh/small-rose/small-rose.github.io/box/font/GuDianMingChaoTi.svg') format('svg');
}
